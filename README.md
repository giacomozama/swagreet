<p align="center">
  swagreet is an aesthetics-focused graphical greetd greeter written in Rust using GTK4 and Relm4.
</p>

## Installation

### Build from Source

```bash
# Clone the repository
git clone https://github.com/giacomozama/swagreet.git
cd swagreet

# Build with Cargo
cargo build --release
```
